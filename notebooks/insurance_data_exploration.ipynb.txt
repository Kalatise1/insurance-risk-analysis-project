## Insurance Data Exploration

This notebook explores a healthcare insurance dataset to understand the factors that influence medical insurance charges.

The analysis focuses on customer characteristics such as age, smoking status, body mass index (BMI), and geographic region. The goal is to identify patterns that may explain variations in insurance costs.

## Create Age Groups

Grouping customers by age helps us better understand how healthcare costs change across different life stages.

```python
df['age_group'] = pd.cut(
    df['age'],
    bins=[18,29,39,49,100],
    labels=['18-29','30-39','40-49','50+']
)

df[['age','age_group']].head()
```

---

## Create BMI Categories

BMI is commonly used to classify body weight into health categories.

```python
def bmi_category(bmi):
    if bmi < 18.5:
        return "Underweight"
    elif bmi < 25:
        return "Normal"
    elif bmi < 30:
        return "Overweight"
    else:
        return "Obese"

df['bmi_category'] = df['bmi'].apply(bmi_category)

df[['bmi','bmi_category']].head()
```
